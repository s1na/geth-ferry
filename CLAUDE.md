# geth-ferry conventions

## Writing style

Do not use em-dashes (—, U+2014) anywhere in this repo: README, docs, code
comments, commit messages, CLI help text. Use commas, colons, parentheses,
semicolons, or new sentences instead. ASCII hyphen-minus (-) is fine.

This is the public-facing tool, so prose should not refer to internal
team workflows, runbooks, or dev-box specifics; describe behavior in terms
any operator can apply.
